# Package index

## All functions

- [`bliss_matrix()`](https://mathilde-robin.github.io/EDITH/reference/bliss_matrix.md)
  : Calculate the additivity matrix according to Bliss' method.
- [`check_packages()`](https://mathilde-robin.github.io/EDITH/reference/check_packages.md)
  : Check and install required packages
- [`checks()`](https://mathilde-robin.github.io/EDITH/reference/checks.md)
  : Checks on the input data matrix.
- [`clean_subtable()`](https://mathilde-robin.github.io/EDITH/reference/clean_subtable.md)
  : Formatting subtable data for analysis.
- [`convert_scientific()`](https://mathilde-robin.github.io/EDITH/reference/convert_scientific.md)
  : Convert to scientific notation.
- [`%>%`](https://mathilde-robin.github.io/EDITH/reference/grapes-greater-than-grapes.md)
  : Import magrittr pipe
- [`index()`](https://mathilde-robin.github.io/EDITH/reference/index-topic.md)
  : Calculate synthetic indexes according to Lehar's method.
- [`install_or_not()`](https://mathilde-robin.github.io/EDITH/reference/install_or_not.md)
  : Install package if not already installed
- [`plot_heatmap()`](https://mathilde-robin.github.io/EDITH/reference/plot_heatmap.md)
  : Plot heatmap for drug combination data
  (data_init/data_bliss/data_diff).
- [`run_EDITH()`](https://mathilde-robin.github.io/EDITH/reference/run_EDITH.md)
  : Main function to run EDITH analysis
- [`save_perm_3drugs()`](https://mathilde-robin.github.io/EDITH/reference/save_perm_3drugs.md)
  : Save permutation results for 3-drug combinations.
- [`save_replicat_2drugs()`](https://mathilde-robin.github.io/EDITH/reference/save_replicat_2drugs.md)
  : Save results for replicates of 2-drug combination experiments.
- [`three_drugs()`](https://mathilde-robin.github.io/EDITH/reference/three_drugs.md)
  : Process data for three-drug combination experiments.
- [`two_drugs()`](https://mathilde-robin.github.io/EDITH/reference/two_drugs.md)
  : Process data for two-drug combination experiments.
