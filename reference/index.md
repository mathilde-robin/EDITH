# Package index

## All functions

- [`bliss_matrix()`](bliss_matrix.md) : Calculate the additivity matrix
  according to Bliss' method.
- [`check_packages()`](check_packages.md) : Check and install required
  packages.
- [`checks()`](checks.md) : Checks on the input data matrix.
- [`clean_subtable()`](clean_subtable.md) : Formatting subtable data for
  analysis.
- [`convert_scientific()`](convert_scientific.md) : Convert to
  scientific notation.
- [`%>%`](grapes-greater-than-grapes.md) : Import magrittr pipe.
- [`index()`](index-topic.md) : Calculate synthetic indexes according to
  Lehar's method.
- [`install_or_not()`](install_or_not.md) : Install package if not
  already installed.
- [`plot_heatmap()`](plot_heatmap.md) : Plot heatmap for drug
  combination data (data_init/data_bliss/data_diff).
- [`run_EDITH()`](run_EDITH.md) : Main function to run EDITH analysis.
- [`save_perm_3drugs()`](save_perm_3drugs.md) : Save permutation results
  for 3-drug combinations.
- [`save_replicat_2drugs()`](save_replicat_2drugs.md) : Save results for
  replicates of 2-drug combination experiments.
- [`three_drugs()`](three_drugs.md) : Process data for three-drug
  combination experiments.
- [`two_drugs()`](two_drugs.md) : Process data for two-drug combination
  experiments.
