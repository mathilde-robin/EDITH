# Calculate synthetic indexes according to Lehar's method.

Calculate synthetic indexes according to Lehar's method.

## Usage

``` r
index(data_init, data_bliss)
```

## Arguments

- data_init:

  A matrix or array containing the initial data.

- data_bliss:

  A matrix or array of the same dimensions as `data_init`, containing
  the calculated Bliss additivity values.

## Value

A list containing the Additivity Index (AI), Combination Index (CI), and
Efficacy Index (EI).
